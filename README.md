Nurtas Portfolio Bot

Telegram-бот-портфолио «Обо мне» — финальный проект курса Middle Python в CapEducation. Бот рассказывает обо мне через 8 разделов, показывает хакатоны с фото и отвечает на свободные вопросы с помощью Gemini AI.

Что внутри


8 разделов портфолио: обо мне, цель, путь в IT, ментор, точка А → точка Б, хобби, лучшие работы, ссылка на GitHub
Раздел «Достижения» — список наград
Раздел «Вайб хакатонов» — список хакатонов, по каждому фото и описание
Свободный вопрос боту → ответ от Gemini AI с контекстом обо мне
Проверка пользовательского ввода через re перед отправкой вопроса в ИИ
Режим --debug через sys.argv


Технологии


Python 3
pyTelegramBotAPI (telebot)
requests
Gemini API (gemini-2.5-flash, ключ из Google AI Studio)
re, sys


Структура проекта

.
├── bot.py             # хендлеры, кнопки, запуск бота
├── content.py         # тексты 8 блоков, достижения, хакатоны
├── ai.py               # запрос к Gemini API
├── requirements.txt
├── .gitignore
└── README.md

Фото для разделов «Лучшие работы» и «Вайб хакатонов» лежат рядом с кодом — частично в корне, частично в подпапках (future_tech/, just_build_it/, AC/, Inspire fest/, STARTUP/), пути точно как указаны в content.py. При установке проекта на новом месте эти файлы нужно перенести в те же пути.

Установка и запуск


Клонировать репозиторий


bash   git clone ВСТАВИТЬ_СЮДА
   cd nurtas-portfolio-bot


Установить зависимости


bash   pip install -r requirements.txt


Запустить бота

также если ии моддель рабботать не будет то можно зайти на https://aistudio.google.com создать ключ и вставить в ai.py и все заработает
bash   python bot.py

С подробным логом входящих сообщений:

bash   python bot.py --debug

Как пользоваться

После /start бот показывает кнопки:

Обо мне · Цель · Путь в IT · Ментор · Точка А -> Точка Б · Хобби · Лучшие работы · GitHub · Достижения · Вайб хакатонов · Задать вопрос ИИ

Любое другое сообщение бот воспринимает как вопрос и отвечает через Gemini AI, основываясь на информации обо мне.
